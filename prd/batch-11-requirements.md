## #1 古诗词默写

**用户故事：** 作为小学生家长，我想让孩子练习古诗默写，检验是否真正记住了诗句。

**与现有"古诗词填空"的区别：**
- 填空 = 挖空关键词，选择/填写单个词
- 默写 = 给上句写下句 / 给下句写上句，手写完整诗句

**功能清单：**
1. 模式选择：给上句写下句 / 给下句写上句 / 随机模式
2. 诗库：小学必背 75 首（复用已有数据）
3. 输入框：textarea 手写输入
4. 即时批改：逐字对比，正确绿色/错误红色/缺字蓝色标注
5. 容错处理：忽略标点差异（"床前明月光" = "床前明月光，"）
6. 繁简体兼容：繁体输入自动转简体比对
7. 统计：正确率 / 已默写数 / 错误最多的诗句
8. 打印：`@media print` 空白默写卷

**验收标准：**
- 75 首诗全覆盖
- 标点容错通过
- 打印排版 A4 正常

**SEO：** "古诗默写, 小学古诗默写, 古诗背诵检测"

## #5 数列求和

**用户故事：** 作为中学生，我想快速计算等差/等比数列的前N项和。

**功能清单：**
1. 两种模式：等差数列 / 等比数列
2. 等差输入：首项 a₁ + 公差 d + 项数 n
3. 等比输入：首项 a₁ + 公比 q + 项数 n
4. 输出：通项公式 + 前N项和公式 + 计算结果 + 公式推导步骤
5. 边界处理：q=1 退化提示 / q 为负数正常计算 / n=0 提示

**验收标准：**
- q=1 边界正确
- 大数不溢出（n≤100）

**SEO：** "数列求和, 等差数列公式, 等比数列"

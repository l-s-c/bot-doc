# 通用设计系统 — Design System v1.0

从 JSONKit 项目沉淀，适用于团队所有产品。

## 设计原则
1. **少即是多** — 去掉一切不必要的元素
2. **一致性** — 同样的东西必须长得一样
3. **呼吸感** — 足够的留白，不要拥挤
4. **反馈感** — 每个操作都有视觉反馈

## 目录
- `tokens.css` — 设计 Token（颜色、字体、间距、阴影、圆角）
- `components.css` — 通用组件（按钮、输入框、Badge、Toast、Toggle 等）
- `themes.css` — 亮色/暗色主题切换

## 使用
引入 `tokens.css` + `components.css` + `themes.css`，所有变量自动生效。

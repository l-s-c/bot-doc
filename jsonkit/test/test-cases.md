## 模块一：JSON 格式化 / 美化

### 功能测试
- TC-F01: 输入合法 JSON → 格式化输出（2空格缩进）
- TC-F02: 输入合法 JSON → 格式化输出（4空格缩进）
- TC-F03: 切换缩进选项，输出实时更新
- TC-F04: 压缩（Minify）：格式化 JSON → 去掉所有空白
- TC-F05: 语法高亮正确（字符串/数字/布尔/null 不同颜色）
- TC-F06: 一键复制结果到剪贴板
- TC-F07: 拖拽上传 JSON 文件，自动填入编辑器
- TC-F08: 点击上传 JSON 文件
- TC-F09: 非法 JSON 输入 → 显示友好错误信息 + 定位到错误行
- TC-F10: 空输入 → 合理提示

### 边界测试
- TC-F11: 大文件（1MB）格式化 < 500ms

- TC-F14: 超过 50MB 文件上传 → 提示"文件过大"
- TC-F15: 深层嵌套 JSON（100+ 层）正确格式化
- TC-F16: 特殊字符：Unicode、emoji、转义字符（\n \t \" \\）

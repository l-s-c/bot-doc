## 开发优先级

**第一批（本周）：** #15-17 + #21（密码生成器、进制转换、单位换算、图片压缩）
**第二批：** #22-24 + #31（图片格式转换、图片尺寸调整、证件照裁剪、二维码生成器）
**第三批：** #32-34 + #39-42（文本工具 + 计算工具，低复杂度批量出）
**第四批起：** 按 SEO 流量数据动态调整优先级

## 原则

- 所有工具纯前端，零后端依赖
- 每个工具独立页面（MPA），独立 SEO 入口
- 复用 shared 组件（Liquid Glass CSS / 导航栏 / 主题切换）
- 文案常量集中管理，预留国际化
- 每个工具 10 轮 QA 测试后上线
